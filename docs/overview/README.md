## What Is Milonexa?

Milonexa is a modular, self-hostable social collaboration platform that combines:

| Category | Features |
|----------|---------|
| **Social** | Feed, posts, friends, follow, reactions, groups, communities, pages |
| **Communication** | Direct messages, group channels, Discord-style servers, voice/video calls |
| **Content** | Videos, blogs, media gallery, uploads, playlists |
| **Streaming** | 60,000+ live TV channels (IPTV), internet radio, live rooms |
| **Meetings** | WebRTC video meetings with 8+ specialized modes |
| **Collaboration** | Documents, wikis, kanban boards, governance tools, knowledge base |
| **Commerce** | Product marketplace, shopping cart, orders, Stripe payments, reviews |
| **AI** | Content moderation, chat assistant, semantic search, smart suggestions |
| **Admin** | 8 admin interfaces for full platform operations |
