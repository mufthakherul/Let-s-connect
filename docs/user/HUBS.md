#### Support Ticket System

For issues you can't resolve yourself:

**Creating a Support Ticket:**

1. Click **"Create Support Ticket"**
2. Select issue type:
   - Account access issues
   - Payment problems
   - Technical bugs
   - Feature requests
   - Billing inquiries
   - Other

3. Describe your issue:
   - Title (subject)
   - Detailed description
   - Steps to reproduce (if applicable)
   - Screenshots/attachments
   - Affected account/product

4. Priority level:
   - Low: General inquiry
   - Medium: Feature not working
   - High: Can't access account
   - Critical: System down / major security issue

**Support Ticket Status:**

- **Open**: Submitted, awaiting review
- **In Progress**: Support team is investigating
- **Waiting on You**: Need more information from you
- **Resolved**: Issue has been fixed
- **Closed**: Ticket archived

**Ticket Tracking:**

```
Ticket #: SUP-2024-001234
Status: In Progress
Priority: High
Created: January 15, 2024
Last Update: 2 hours ago

Issue: Unable to reset password
Support Response:
  Thank you for contacting Milonexa Support. We've
  received your ticket. A specialist will review your
  issue and respond within 24 hours.

[Reply to Ticket] [View Full History]
```

**Response Times:**

- **Critical**: Response within 1 hour
- **High**: Response within 4 hours
- **Medium**: Response within 24 hours
- **Low**: Response within 48 hours

#### Reply to Support Tickets

1. Go to your ticket
2. Click **"Reply"**
3. Add your message
4. Attach files if needed
5. Click **"Send Reply"**

Responses from support team appear in reverse chronological order (newest first).

#### Upvoting Posts

Help others find useful contributions:

- **Upvote**: Click the ⬆️ arrow to upvote
- **Downvote**: Click the ⬇️ arrow (for non-constructive posts)
- **Saved**: Click bookmark to save for later

Posts with more upvotes appear higher in the thread.

#### Forum Search

Find previous discussions:

1. Click **Search Forums**
2. Enter keywords
3. Filter by:
   - Category
   - Time period (last day, week, month, year)
   - Solved posts only
   - Posted by [user]

#### Forum Etiquette

Helpful discussion tips:

✓ **Do:**
- Search before posting (avoid duplicates)
- Use clear, descriptive titles
- Format your post (use lists, code blocks, etc.)
- Include relevant details
- Stay on topic
- Be respectful and constructive
- Use English (or forum's language)
- Search for solutions before posting

✗ **Don't:**
- Post spam or promotional content
- Spam or flood threads
- Attack other users
- Post personal information
- Share sensitive data (passwords, API keys)
- Post copyrighted content
- Share off-topic content

#### Moderators & Community Helpers

Community moderation is handled by:

- **Milonexa Staff** (red badge): Official team members
- **Moderators** (green badge): Trusted community members
- **Community Helpers** (blue badge): Active, helpful contributors

You can report problematic posts to moderators.

## Accessibility Hub

### Overview

Resources for accessible use of Milonexa.

**Access:** `/hubs/accessibility`

### WCAG Compliance

Milonexa complies with WCAG 2.1 Level AA standards:

- ✓ Perceivable: Content is accessible to all senses
- ✓ Operable: Keyboard and voice control support
- ✓ Understandable: Clear language and organization
- ✓ Robust: Compatible with assistive technologies

### Accessibility Features Guide

#### Visual Accessibility

- Dark/light/system modes
- High contrast mode
- Text scaling (80-150%)
- Dyslexia-friendly fonts
- Reduced motion option

#### Audio Accessibility

- Auto-captions on videos
- Transcripts for audio content
- Audio descriptions
- Sound-based alerts also have visual alerts

#### Motor Accessibility

- Full keyboard navigation
- Voice control support
- Customizable key bindings
- Switch control compatibility

#### Cognitive Accessibility

- Clear language
- Simplified mode option
- Logical content organization
- Consistent navigation

### Report Accessibility Issues

Found an accessibility problem?

