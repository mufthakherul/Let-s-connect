## Feature Highlights

- 📱 **Social Feed** — Post, comment, react, share (Facebook/Twitter-style)
- 💬 **Real-time Chat** — Direct messages, channels, voice/video calls
- 📺 **Streaming** — 60,000+ live TV channels, internet radio
- 🛒 **Shop** — Browse and purchase products
- 📄 **Docs & Wiki** — Collaborative documents, wikis, project boards
- 👥 **Groups** — Public, private, and secret groups
- 🔖 **Bookmarks** — Save content for later
- 🤖 **AI Features** — Content moderation, smart suggestions
