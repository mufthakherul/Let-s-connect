# Getting Started

## Local setup (quick path)

1. Clone repository
2. Copy `.env.example` to `.env`
3. Start required infrastructure (`postgres`, `redis`)
4. Start frontend and required backend services

## Recommended onboarding path

- Read docs hub: `docs/README.md`
- Review architecture: `docs/overview/ARCHITECTURE.md`
- Follow setup guide: `docs/development/SETUP.md`
- Review deployment quick start: `docs/deployment/QUICK_START.md`
- Read testing index: `TESTING.md`

## First-day checklist

- [ ] Run frontend locally
- [ ] Run at least one backend service locally
- [ ] Verify API gateway health endpoint
- [ ] Run frontend unit tests
- [ ] Run critical-path baseline script
