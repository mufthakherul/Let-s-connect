# Release Process

This page summarizes release planning and rollout practices.

## Stages

1. Scope and risk assessment
2. CI quality gate pass
3. Staging verification
4. Production rollout
5. Post-release validation
6. Release notes publication

## Pre-release checklist

- CI checks green
- migration notes prepared
- rollback path confirmed
- owner + on-call awareness confirmed
