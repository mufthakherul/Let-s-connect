# Milonexa Mobile (Expo)

React Native + Expo scaffold for iOS and Android.

## Quick start

1. Install dependencies in this folder.
2. Start Expo dev server.
3. Run on iOS simulator, Android emulator, or physical device.

Set `EXPO_PUBLIC_API_URL` to your API gateway host (default is `http://localhost:8000`).

## Shared API layer

This mobile app imports `frontend/src/shared/apiClient.js` to keep a shared web/mobile API client contract.
