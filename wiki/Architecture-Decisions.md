# Architecture Decisions (ADR)

Architecture Decision Records capture why key technical choices were made.

## ADR workflow

1. Create ADR from template.
2. Review in PR with affected owners.
3. Mark as Accepted when merged.
4. Supersede old ADRs with references when replaced.
