## 1. Direct Messages

### Starting a Conversation

**Method 1: Via Messages tab**
1. Click the **💬 Messages** icon in the navigation.
2. Click **+ New Message** (top-right of the panel).
3. Search for a user by name or username.
4. Click their name to start a conversation.
5. Type your first message and hit **Enter** or click **Send**.

**Method 2: Via user profile**
1. Visit the user's profile.
2. Click the **Message** button.
3. The conversation window opens immediately.

### Conversation List

The Messages panel shows all your conversations sorted by most recent. Each conversation shows:
- The other user's profile picture
- Preview of the last message
- Timestamp
- Unread message count (blue badge)

### Message Status Indicators

| Icon | Meaning |
|------|---------|
| ✓ (single tick) | Message sent |
| ✓✓ (double tick) | Message delivered |
| ✓✓ (blue ticks) | Message read |

> 💡 You can hide your read receipts in **Settings → Privacy → Read Receipts**.

## 2. Group Conversations

Create a group conversation with multiple people.

### Creating a Group Conversation

1. Go to **Messages → + New Message**.
2. Search and add **2 or more** participants.
3. Optionally add a **group name** and **group icon**.
4. Click **Create Group**.

### Managing Group Conversations

As the conversation creator:
- Rename the group (tap the group name at the top)
- Change the group icon
- Add new members
- Remove members
- Leave the group

**Group size limit:** Up to **250 participants** in a group conversation.

## 3. Message Features

### Text Messages

- Up to **4,000 characters** per message
- Markdown-style formatting: **bold**, *italic*, `code`, ~~strikethrough~~
- @mention participants in group conversations
- Emoji support (inline emoji picker with 🔍 search)

### Reactions to Messages

Hover over (or long-press) any message → click a reaction emoji.

### Replies & Threads

- Click **Reply** on any message to reply in context
- The original message is quoted in your reply for clarity

### Editing Messages

- You can edit a sent message within **5 minutes** of sending
- Edited messages are labelled *(edited)*
- Edit: hover over message → ✏️ Edit

### Deleting Messages

- Delete your own messages at any time
- Deleted messages are replaced with *"This message was deleted"*
- Delete: hover → 🗑️ Delete

### Pinning Messages

In group conversations and servers:
- Hover over a message → 📌 Pin
- Pinned messages are accessible from the top of the conversation
- Up to **50 pinned messages** per conversation

### Scheduled Messages

Send a message at a future time:
1. Type your message.
2. Click the **⏰** icon next to the send button.
3. Choose date and time.
4. Click **Schedule**.

## 4. File Sharing in Messages

| Type | Max Size | Formats |
|------|---------|---------|
| Images | 25 MB each | JPG, PNG, WebP, GIF |
| Videos | 100 MB | MP4, WebM, MOV |
| Audio | 25 MB | MP3, WAV, OGG |
| Documents | 50 MB | PDF, DOC, DOCX, XLS, XLSX, PPT, ZIP, etc. |

### How to Share Files

1. In the message input, click the **📎 Attach** button.
2. Select a file from your device.
3. Add an optional caption.
4. Click **Send**.

### Media Gallery

Every conversation has a **Media** tab (📷 icon at the top) that shows all shared images and files in chronological order, making it easy to find shared content.

## 5. Community Servers

Community Servers are spaces for larger groups — similar to Discord. Each server has **channels** for different topics.

### Joining a Server

1. Get a **server invite link** from a server member or admin.
2. Visit the invite link and click **Join Server**.
3. The server appears in your sidebar.

### Creating a Server

1. Click the **+** icon in the server list sidebar.
2. Choose **Create Server**.
3. Give your server a name and icon.
4. Configure the default channels.
5. Invite members via the generated invite link.

### Server Channels

| Channel Type | Purpose |
|-------------|---------|
| **# Text channel** | Text messages, links, images, files |
| **🔊 Voice channel** | Live voice room — join to talk |
| **📢 Announcement channel** | Server-wide announcements (admins only) |

### Server Roles

Server admins can create roles to manage permissions:

| Default Role | Permissions |
|-------------|-------------|
| **Owner** | Full control over the server |
| **Admin** | Manage channels, members, roles |
| **Moderator** | Delete messages, kick/ban members |
| **Member** | Read and post in allowed channels |
| **Guest** | Read-only in designated channels |

## 6. Voice & Video in Messages

### Voice Calls (1-to-1)

1. Open a direct message conversation.
2. Click the **📞 Call** button (top-right).
3. The other user receives an incoming call notification.
4. They accept — the call connects.

### Video Calls (1-to-1)

1. Open a direct message conversation.
2. Click the **📹 Video** button.
3. Grant camera and microphone permissions if prompted.
4. The other user receives an incoming video call notification.

### Voice Channels in Servers

Server voice channels are **always-on** rooms — you join and leave at will:
1. Click on a voice channel (🔊) in the server sidebar.
2. Your microphone is active automatically (mute with the mic icon).
3. Others in the channel can hear you; you can hear them.
4. Click the ✕ or ☎️ Disconnect button to leave.

### Controls During Calls

| Control | Description |
|---------|------------|
| 🎤 Mute/Unmute | Toggle your microphone |
| 📹 Camera on/off | Toggle your camera |
| 🖥️ Screen Share | Share your entire screen or a window |
| 🔊 Deafen | Stop hearing others (and hide your audio) |
| ☎️ End Call | Leave the call |

## 7. Message Settings & Privacy

### Who Can Message You

Control who can initiate conversations: **Settings → Privacy → Who can message me**.

| Setting | Effect |
|---------|--------|
| **Everyone** | Any registered user can message you |
| **Friends only** | Only mutual friends can message you |
| **Nobody** | No direct messages (useful for public figures) |

### Read Receipts

Turn off read receipts: **Settings → Privacy → Read Receipts → Off**.

> Note: If you turn off read receipts, you also won't see read receipts from others.

### Active Status

Hide your online/active status: **Settings → Privacy → Activity Status → Nobody**.

### Message Requests

Messages from people who aren't your friends go into **Message Requests**. You must accept a request to be able to reply.

## 8. Push Notifications for Messages

Configure message notifications: **Settings → Notifications → Messages**.

| Option | Description |
|--------|------------|
| **All messages** | Notify for every new message |
| **Mentions only** | Only notify when @mentioned |
| **Off** | No message notifications |

You can also mute individual conversations:
1. Right-click (or long-press) a conversation.
2. Select **Mute** → choose duration (1 hour, 8 hours, 24 hours, indefinitely).

## 9. Blocking & Reporting in Messages

### Blocking from a Conversation

1. Open the conversation.
2. Click the user's name or **ⓘ Info** at the top.
3. Click **Block @username**.

Blocking prevents the user from seeing your profile or sending further messages.

### Reporting a Message

1. Hover over the message → click **⋮ More**.
2. Select **Report**.
3. Choose a reason:
   - Harassment or bullying
   - Spam
   - Inappropriate or explicit content
   - Threats or violence
   - Other
4. Add optional context and submit.

Reports are reviewed by the Trust & Safety team.
